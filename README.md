# mkcold-archive

本仓库用于归档与发布多个独立的 HTML 页面。这些页面以静态文件的形式存储，不针对任何特定域名或网站，仅作为资料留存与访问之用。

## 项目简介

mkcold-archive 是一个通用型 HTML 页面归档仓库。所有页面均为独立文件，可直接在浏览器中打开查看。仓库内容不定期更新，用于保存历史版本或特定用途的页面快照。

## 目录说明

```
.
├── index.html          # 仓库首页 / 归档列表
├── pages/              # 归档页面存放目录
│   ├── example-1.html
│   └── example-2.html
└── assets/             # 公共资源（图片、样式、脚本等）
    ├── style.css
    └── script.js
```

- `pages/` 目录下存放各个归档 HTML 文件，文件名通常反映页面内容或归档时间。
- `assets/` 目录用于存放页面引用的公共资源文件，便于统一管理。

## 页面归档说明

- 每个归档页面均为完整的 HTML 文件，可独立运行。
- 页面内容可能包含文本、样式、脚本等，均内嵌或引用自本仓库资源。
- 归档文件命名格式无强制要求，但建议使用描述性名称以便识别。
- 仓库不承诺所有页面永久有效，可能因维护需要删除或更新部分文件。

## 维护说明

- 本仓库由维护者不定期更新归档内容。
- 若需提交新的归档页面，请通过 Pull Request 提交，并确保文件不包含恶意代码或敏感信息。
- 欢迎提出 Issues 反馈问题或建议，但请注意本仓库不提供任何形式的服务承诺。

## 许可

本仓库内容仅供学习与参考使用，不提供任何明示或暗示的担保。请勿将归档页面用于违反法律法规或侵犯他人权益的用途。
